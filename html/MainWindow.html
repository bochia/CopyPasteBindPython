NOTE - dont copy text directly from within these textboxes, always use hotkeys to get text. These textareas are only for viewing - text has had newline characters removed.


HOW TO USE:

Bind Text To Number - highlight the text you want to copy and then use one of the hotkey(s) below.

  • ctrl + c + number key

Paste Binded Text - use one of the hotkey(s) below.

  • ctrl + b + number key
  • Use function keys fn13 - fn22 to paste text from numbers 0 - 9 respectively.

This window must stay open to use the CopyPasteBind functionality, but feel free to minimize it.

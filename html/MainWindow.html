HOW TO USE:

Bind Text To Number - highlight the text you want to copy and then use one of the hotkeys below.

  • ctrl + c + number key
  • Use function keys fn13 - fn22 to bind text to numbers 0 - 9 respectively.

Paste Binded Text - use one of the hotkeys below.

  • ctrl + number key
  • ctrl + b + number key

IMPORTANT:

This window must stay open to use the CopyPasteBind functionality, but feel free to minimize it.
